# Crypto Reports

Daily crypto market intelligence — generated by Riven (AI) at ~7AM MYT.

**Live:** https://crypto-reports-cc8.pages.dev/

## How It Works

- Cron job runs daily at 23:00 UTC (7:00 AM MYT)
- Generates crypto report from live data (BTC, ETH, DeFi, Fear & Greed, news, catalysts)
- Saves to `09_project-summaries/daily/crypto-research-YYYY-MM-DD.html` in obsidian-vault
- Copies to this repo and deploys to Cloudflare Pages

## Report Contents

- BTC price, MVRV ratio, SOPR indicator
- ETH price, ETH/BTC ratio
- Fear & Greed index, BTC Dominance
- 7-day altcoin performance
- DeFi TVL snapshot
- Stablecoin flows
- Market sentiment
- Bullish/Bearish/Neutral news
- Key catalysts
- One-liner summary
